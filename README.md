# Resume Analyzer

## 📌 Project Overview

Resume Analyzer is a beginner-level Java application that analyzes a resume and provides feedback based on important resume sections such as contact information, education, skills, experience, and keywords. The application calculates a score and provides suggestions to improve the resume.

## 🚀 Features

* Checks for contact information (Email, Phone Number, LinkedIn)
* Verifies education details
* Detects technical and soft skills
* Analyzes work experience and projects
* Counts professional resume keywords
* Calculates an overall resume score
* Provides recommendations for improvement

## 🛠️ Technologies Used

* Java
* Scanner Class
* ArrayList
* Object-Oriented Programming Concepts

## 📂 Project Structure

ResumeAnalyzer.java

## ▶️ How to Run

### Prerequisites

* Java JDK 8 or higher
* Visual Studio Code (or any Java IDE)

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/resume-analyzer.git
   ```

2. Navigate to the project directory:

   ```bash
   cd resume-analyzer
   ```

3. Compile the program:

   ```bash
   javac ResumeAnalyzer.java
   ```

4. Run the program:

   ```bash
   java ResumeAnalyzer
   ```
